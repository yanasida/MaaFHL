## MAAFHL 忘川风华录

### start
```txt
python -m pip install maafw==4.4.1
```

## 功能列表

#### 桃源居

体力领取、待办、分享、考工治图、随机家具

#### 日常

速办、知交圈点赞、赠领友情点、邮件、喵居吸猫喂猫、世界对话2次

#### 雅舍

河灯（默认第一个）、捐赠

#### 金戈清符

目前只支持九段及以下（没有号，上不去）,请提前准备好阵容

#### 沙盘清符

可选择购买次数,请提前准备好阵容

不定期更新,主要是懒
